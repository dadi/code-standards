# DADI code standards

Configs for linters and formatters used in DADI projects.

## Usage

### Prettier

Install:

`npm i -D prettier @dadi/prettier-config`

In `.prettierrc`:

```javascript
'@dadi/prettier-config'
```

### Eslint

Install:

`npm i -D eslint @dadi/eslint-config`

In `.eslintrc`:

```javascript
{
  "extends": "@dadi"
}
```
